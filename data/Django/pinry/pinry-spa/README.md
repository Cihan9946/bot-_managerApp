# pinry-spa

## Project setup
This project requires `NodeJS@18`.

```
npm install -g "pnpm==9.11.0"
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm serve
```

### Compiles and minifies for production
```
pnpm build
```

### Lints and fixes files
```
pnpm lint
```
